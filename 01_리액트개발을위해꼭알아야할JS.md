# CH1. 리액트 개발을 위해 꼭 알아야할 자바스크립트

## 1.1 자바스크립트의 동등 비교

### 1.1.1 JS의 데이터 타입

- 원시 타입

  - boolean / null / undefined / number / string / symbol / bigint

- 객체 타입

  - object

- 원시 타입과 객체 타입의 차이
  - 원시 타입은 메모리에 값 자체를 저장
  - 객체 타입은 프로퍼티를 수정할 수 있으므로 원시 타입과 다르게 변경 가능한 형태로 저장, 값을 복사 할 때도 참조 값이 복사
  - 단, 객체 내부의 프로퍼티의 값은 원시 형태로 저장
